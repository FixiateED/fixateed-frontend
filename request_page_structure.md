# بنية صفحة الطلبات

## الخطوات الثلاث
1. **معلومات الجهاز** (الخطوة الحالية)
   - اختيار الشركة المصنعة (آبل، سامسونج، هواوي، ديل، إتش بي، لينوفو)
   - اختيار الموديل (بعد اختيار الشركة)
   - اختيار المشكلة/الخدمة المطلوبة

2. **الموقع والتواصل**
   - معلومات الاتصال
   - العنوان/الموقع
   - تفاصيل إضافية

3. **التأكيد**
   - مراجعة البيانات
   - إرسال الطلب
